# Books Fullstack Application

Books fullstack application developed using Nest.js and Next.js 13.


## Backend Tech stack
 - Nest.js
 - PostgreSQL
 - TypeORM
 - Swagger API documentation

## Frontend Tech stack
- Next.js 13
- TailwindCSS
- React Query
- Axios

## Features
- Atomic Design
- Infinite Scroll
- Pull to refresh
- Responsive Design
